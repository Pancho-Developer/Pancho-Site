Recursos

Como desarrolladores siempre buscamos la manera mas sencilla de hacer nuestros proyectos, por ello, ciertas personas han hecho sitios web con increíbles recursos que podemos usar para facilitar nuestro trabajo.

Google Fonts

Seguramente siempre has buscado la fuente especifica para tu Sitio Web, pues con Google Fonts puedes encontrar la ideal. Google Fonts tiene un gran numero de fuentes de libre uso, solo debes seleccionar la que mas te guste y te dara la opcion de copiar la etiqueta link, o el import en el CSS de tu web. Con esto solo deberas seleccionarlo como Fuente predeterminada si es que la quieres para toda tu web, sin duda, una muy buena y util.



Font Awesome

Font Awesome es un conjunto de herramientas de fuentes e iconos basado en CSS y Less. Fue hecho por Dave Gandy para su uso con Bootstrap, y luego se incorporó a BootstrapCDN.
